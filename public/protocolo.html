Protocolo Dysrupcion

Despliega tu propia comunidad tech en 30 minutos

01

Fork this repo

Haz un fork del repositorio en tu cuenta de GitHub. Esto crea tu propia copia del protocolo.

gh repo fork dysrupcion/dysrupcion --clone
02

Clone and install

Clona tu fork e instala las dependencias del proyecto.

cd dysrupcion npm install
03

Edit config

Personaliza la configuracion de tu comunidad editando el archivo de configuracion principal.

# wrangler.toml o config.ts COMMUNITY_NAME = "Tu Comunidad" MISSION = "Tu mision aquí" COLORS = { pink: "#FF1654", cyan: "#2EECC7", yellow: "#FFD100" } TRACKS = ["Producto", "Ingenieria", "Diseno", "Datos", "DevOps", "Comunidad"]
04

Set up Supabase

Crea un proyecto en Supabase y conecta tu base de datos. Sigue la guia completa para configurar las tablas y permisos.

# Copia las variables de entorno cp .env.example .env # Agrega tus credenciales de Supabase SUPABASE_URL=https://tu-proyecto.supabase.co SUPABASE_ANON_KEY=tu-anon-key

Guia completa de Supabase →

05

Deploy to Cloudflare

Despliega tu sitio a Cloudflare Workers. Tu comunidad estara en linea en segundos.

# Login a Cloudflare npx wrangler login # Deploy npx wrangler deploy
